C8yConnection¶

class apamax.eplapplications.connection.C8yConnection(url, username, password)[source]¶

Simple object to create connection to Cumulocity IoT and perform REST requests.

Parameters

  • url – The Cumulocity IoT tenant url.

  • username – The username.

  • password – The password.

request(method, path, body=None, headers=None, useLocationHeaderPostResp=True)[source]¶

Perform an HTTP request. In case of POST request, return the id of the created resource.

Parameters

  • method – The method.

  • path – The path of the resource.

  • body – The body for the request.

  • headers – The headers for the request.
